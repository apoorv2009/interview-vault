# Tree DFS/BFS

**Concept:** Depth-first (recursive/stack) or breadth-first (queue) traversal of trees.

**When to use:**
- Tree height, path sum, level order, diameter, LCA

**Complexities:** Time O(n) | Space O(h) DFS, O(w) BFS
